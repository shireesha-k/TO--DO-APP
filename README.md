# ✅ TO-DO APP

## 📌 Description

This is a simple TO-DO App created using HTML, CSS, and JavaScript. It allows users to add, mark as completed, and delete tasks from their list.

## ✨ Features

* ➕ Add tasks to the list
* ✅ Mark tasks as completed
* ❌ Delete tasks

## 💡 Technologies Used

* 🌐 HTML
* 🎨 CSS
* ⚡ JavaScript
